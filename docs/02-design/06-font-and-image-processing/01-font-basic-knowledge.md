# 字体类型







## sans-serif

表示字体不是无衬线字体。无衬线字体（如 Arial）是没有小装饰线条（衬线）的字体，这里表示当前字体不是无衬线字体。

## fixed

表示字体不是等宽字体（Fixed-Width Font）。等宽字体（如 Courier New）每个字符占据相同的水平空间，这里表示当前字体不是这种类型。

## serif

表示字体不是有衬线字体。有衬线字体（如 Times New Roman）在字符的末端有小装饰线条，这里表示当前字体不是有衬线字体。

## LCD

表示这是适用于 LCD 屏幕的字体，通常用于小尺寸显示器或电子设备，优化显示效果。

1. **`'icon': true`**
   表示这是一种图标字体（如 FontAwesome），通常用于显示图标而非文字。
2. **`'custom': false`**
   表示这不是一个自定义字体，而是系统或预定义字体。
