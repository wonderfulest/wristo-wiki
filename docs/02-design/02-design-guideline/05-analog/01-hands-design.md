# 🌟 一、表盘指针的核心结构（必须明确）

指针其实是 **三要素**：

1. **Pivot（旋转点）**：所有指针围绕的点，必须精准。
2. **Body（指针主体）**：长条、骨架、装饰、渐变等。
3. **Counterweight（配重）**：可选，美学用。

在 Figma 要做到的，就是让 *所有内容都围绕 pivot 旋转，而不是围绕边界框*。

------

# 🌙 二、画布设置（避免后期错位）

### ① 新建画框 Frame

- 尺寸：`454×454`（AMOLED）或 `280×280`（MIP）
- 名称：`Watch-Canvas`

### ② 在中央创建 pivot

- 使用 Ellipse
- 大小：`6×6`（不透明 100%）
- 填充黑色/白色皆可，只是为了定位
- 放在 **绝对中心（选中后按：Shift + H + V）**

这个点后面会隐藏掉，它只是“宇宙中心”。

------

# 🔧 三、绘制时针（Hour Hand）

让我们给它一点仪式感 🌌：

### ① 起笔方式

- 使用 **Pen** 工具画出一条从 pivot 向上的粗线段。
- 不要太长（占整表盘半径的 40–50%）。

### ② 线条属性

- Stroke: `8–14 px`
- Cap: Round
- Join: Round

### ③ 美感增强

可以根据风格加入：

- 矩形 + 蒙版，形成平头指针
- 凹槽、渐变金属效果（Fill → Linear）
- 中空骨架（Boolean Subtract）

### ④ **设定旋转中心**

⚠️ 这是关键
 把指针整体包成一个 Group：

- 选中指针 → `⌘G`
- 在右侧 → `Angle` 旁边点 “圆点图标”
- 把旋转中心点拖到 **Pivot 的正中心**

> 之后无论怎么缩放、复制、变形，旋转永远不会飘。

------

# ⚡ 四、绘制分针（Minute Hand）

它是舞台上更轻盈的那位：

### 风格建议

- 与时针风格一致，但更细、略长
- 长度：占表盘半径的 60–70%
- 粗细：`6–10 px`
- 可加入金属亮条或透明小段，让它更有科技感

### 同样设置 pivot

与时针完全一致：放到中心旋转点上。

------

# 🔥 五、绘制秒针（Second Hand）

这是整个表盘的“动感之魂”。可以加入更多戏剧性元素：

### 建议样式

- 极细：`2–4 px`
- 鲜艳：红、金、白（依主题）
- 末端可以设计：小圆点、镂空圆、滴答形状

### 配重 Counterweight

在 pivot 下方来一个小球或椭圆形，形成视觉平衡。
 许多高端机械表都有这个结构，效果极好。

### 同样设置 pivot

保持一致性。

------

# 🌈 六、让三个指针保持完美对齐

在 Figma 做表盘时最容易错的点就是 *指针 anchor 对不齐*。

### 最靠谱的方法：

1. 把 **时针、分针、秒针全部放进一个 Frame**
2. 在 Frame 里放一个辅助 **Pivot 点**
3. 每个指针的旋转点都贴到这个 pivot 上
4. 调整排列顺序：
   - 最底层：时针
   - 中间：分针
   - 顶层：秒针

------

# ❄️ 七、导出规范（非常关键）

为了让你的 Garmin 表盘旋转时无偏移：

### 必须用 **PNG，大小与画布一致**

例如：

- Watch-Canvas 为 454×454
- 时针 PNG 也必须导出成 **454×454**（透明背景）

指针本体虽然只占很小部分，但导出的 PNG 不能裁边！

**否则旋转会错位。**

# 🧩 八、常见指针风格模板（你可以直接在 Figma 复用）

你可以从这几类风格开始尝试：

### ● Minimal 极简风

- 无渐变
- 单色
- 几何造型
- 非常适合 MIP

### ● Elegant 雅致风

- 金属拉丝渐变
- 指针头带切面
- 细边框 + 透明中段

### ● Sport 运动风

- 高对比色（亮橙、亮蓝）
- 秒针带长亮条
- 视觉动感更强

### ● Retro 复古机械风

- 大配重
- 三角形指针头
- 古铜色、奶白色、浮雕线
