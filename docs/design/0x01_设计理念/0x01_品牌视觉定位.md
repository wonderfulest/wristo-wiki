# **Wristo｜品牌视觉定位文档（V1.0）**

## 0. 品牌核心一句话

**“Wristo — 当时间遇见审美。”**
 科技织底，设计发光，让每一块手腕，都拥有属于自己的风格宇宙。

------

# 1. 品牌愿景（Vision）

打造全球最可信赖、最具美学品味的 **智能表盘设计生态**，让数百万用户能自由表达风格、掌控每日节奏，享受穿戴科技的温度。

------

# 2. 品牌使命（Mission）

**让表盘不只是工具，而是用户个性的投影。**

通过技术、设计与生态融合，为全球 Garmin 用户提供：

- 高品质审美设计
- 功能可靠的数据展示
- 多语言、跨区域的购买体验
- 持续演进的 watch face 创作平台

------

# 3. 品牌个性（Personality）

| 维度         | 特质定义                                             |
| ------------ | ---------------------------------------------------- |
| **美学审慎** | 神经末梢敏锐，保持高级画面感与品味判断。             |
| **工程可靠** | 底层结构稳固，性能与体验都不做妥协。                 |
| **温暖友好** | UI / 文案 / 沟通方式都不生硬，像一位细致的设计好友。 |
| **全球一致** | 多语言、多文化、多地区场景下保持统一视觉秩序。       |
| **持续创新** | 保持更新频率与产品活力，让用户感受到不断生长。       |

------

# 4. 目标用户（Audience）

Wristo 的用户分为三类，每类都有不同的视觉偏好：

### ① **日常佩戴者（主力：欧美 + 女性用户占比高）**

- 喜欢花卉、自然、干净背景
- 对审美敏感
- 愿意付费购买高颜值表盘
- 偏好淡雅、柔光、极简风格

### ② **运动用户（主力：美国 / 欧洲男性为主）**

- 偏好强对比、黑金、机械感、炫光线条
- 注重数据可读性、功能性
- 喜欢有氛围但不喧宾的设计

### ③ **设计师与创作者（Studio 生态的未来用户）**

- 关注规则、组件、图层结构
- 需要统一规范指导自己的作品
- 未来将对 Wristo 是生态贡献者

------

# 5. 视觉主张（Visual Essence）

**“稳态科技 + 轻盈美学”**

稳态科技：

- 信息展示精准
- 系统感、结构感明确
- 颜色不喧哗、界面克制

轻盈美学：

- 背景干净、氛围细腻
- 色彩柔和但有记忆点
- 花卉、光斑、氛围渐变等视觉语言可作为 signature

------

# 6. 品牌主视觉元素（Visual Assets）

## 6.1 Logo 使用逻辑

- **Logo 基础款**用于 App、官网、Studio 产品
- **极简字标（WRISTO）**用于启动页、文案主标题
- **单色版**用于 Dark/Light 的对比场景

Logo 气质关键词：
 **精确、稳重、可扩展、独立感**

------

# 6.2 色彩体系（Color System）

Wristo 采用 **三层色彩架构**：

### **主品牌色（Brand Colors）**

| 名称           | Hex     | 气质                   |
| -------------- | ------- | ---------------------- |
| Wristo Blue    | #2E4CFF | 可信赖、专业、科技感   |
| Midnight Black | #0F0F17 | 高级感、沉稳、背景主力 |
| Cloud White    | #FAFAFA | 纯净、留白、内容呼吸   |

### **扩展色（Accent Colors）**

| 色号    | 描述               |
| ------- | ------------------ |
| #EAC6F9 | 花卉自然系的柔光紫 |
| #F9D9D9 | 温柔粉香调         |
| #FFD56F | 活力暖光           |
| #54C1FF | 清爽天蓝           |

### **功能色（Functional Colors）**

| 功能    | 色号    | 说明           |
| ------- | ------- | -------------- |
| Success | #2ECC71 | 激活、成功状态 |
| Warning | #F1C40F | 异常提醒       |

> 设计原则：**避免饱和度过高；强调柔光、透明度、渐变。**

------

# 6.3 字体（Typography）

### **品牌主字体**

- **Noto Sans SC**（中英双语言一致性最佳）
- 标题字重：600–700
- 正文字重：400–500

### **表盘字体**

- 图形数字字体（mono 系列）
- 跨语言字体（CJK / Multilingual）
- 可扩展为自研子系列：**Wristo Numeric / Wristo Iconic**

------

# 6.4 图形语言（Graphic Style）

**花卉、柔光、渐变、自然纹理**是 Wristo 的 signature。

可以出现在：

- 表盘背景
- 宣传海报
- App promotional banner
- 官网 hero 图

风格关键词：
 **柔光、空气感、空镜头、淡淡冷色边缘、自然弧线、细腻颗粒**

------

# 6.5 品牌摄影方向（Photography Direction）

### 主题

**“自然的诗，佩戴的光。”**

可以围绕：

- 花瓣、树影、空气流动
- 阳光、镜头眩光、柔焦
- 运动场景的动感瞬间
- 手腕的特写（干净背景）

摄影色调：
 **高级灰、淡紫、浅粉、雾蓝、淡金、黑金**

------

# 7. 品牌情绪板（Moodboard）

视觉氛围可聚焦四类：

1. **Elegant Flora（花影氛围）**
2. **Black & Gold Luxury（科技黑金）**
3. **Sporty Neon（运动光线）**
4. **Clean Tech Minimal（极简科技）**

可作为今后表盘系列与主题分类的主线。

------

# 8. 品牌应用示例（Applications）

### 官网 / App

- 首屏使用大幅 Hero Image（花卉 / 科技渐变）
- CTA 按钮以 Wristo Blue 为主
- 产品卡片采用轻底色 + 圆角 + 柔光

### Watch Face Store

- 保持系列命名统一
- 产品封面要强化**品牌调性，而非设计师⾵格混乱**

### 社交媒体

- 保持一致滤镜
- 使用“高光柔光 + 简短文案”的封面风格

### Studio 创作平台

- 使用干净背景、功能性高的线框 UI
- 保持 Wristo 的科技稳定气质

------

# 9. 禁用规范（Don’ts）

为了避免视觉跑偏，以下内容需**严格禁止**：

- 过度饱和的高亮色
- 低质颗粒、AI 生成痕迹明显的素材
- 画面元素堆叠、信息爆炸
- 大面积渐变彩虹效果
- 表盘封面与品牌官网风格不一致

------

# 10. 品牌未来演进方向（Roadmap）

Wristo 的视觉语言将不断进化，未来重点包括：

- 自研字体 **Wristo Numeric**、**Wristo Iconic**
- 系列化的表盘视觉规范（花卉 / 黑金 / 运动 / 极简）
- 官网全套品牌组件库（Design Tokens + UI Library）
- App 视觉统一化（多语言布局优化）
- 全球品牌内容统一（博客、邮件、社交媒体）
- Studio 设计生态的视觉标准化
