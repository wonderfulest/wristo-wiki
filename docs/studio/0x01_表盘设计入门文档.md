# 设计以及开发步骤

## 找图片

去图片网站（不限制），下载一张 png 格式的图片作为背景。

我经常用的就是 https://www.freepik.com/ ， 当然也可以在其他地方查找合适图片。



**版权问题：**避免使用商用图片，如果必须使用，可以使用 photoshop或者AI 工具进行二次创作，可以避开商业版权纠纷。当然，前提是我们挣到了很多钱，值得对方请律师从美国来中国打官司。

## 编辑图片

将上一步下载好的图片，编辑成600x600像素 以上大小的**正方形png格式**。

推荐使用 https://www.fotor.com/ 进行处理。

编辑完成后，下载保存成  **background-0.png**，如下面示例：



<img src="https://muyids.oss-cn-beijing.aliyuncs.com/img/image-20240603234019399.png" alt="image-20240603234019399" style="zoom:50%;" />

## 编辑表盘

访问表盘设计工具网站：http://47.97.4.185/watchface-builder-vue/index.html，

摆放背景图片、时间（时\分\秒\24H）、日期、电池、指示器、数据项等元素。如下图：

<img src="https://muyids.oss-cn-beijing.aliyuncs.com/img/image-20240603234212526.png" alt="image-20240603234212526" style="zoom: 25%;" />

并导出 json 文件，文件名为 conf.json
