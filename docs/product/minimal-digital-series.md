# 🕶️ **2026 男性向「极简 & 数字」表盘全年上新计划**

------

# 🎯 核心定位（2026 男性用户趋势）

男性用户偏好：

- **大数字**（强可读性）
- **高对比度（黑白 / 黑橙 / 黑青色）**
- **极简构图**（留白、几何）
- **功能感强**（数字、线条、网格）
- **科技边缘感**（霓虹、HUD、光边）
- **稳定、不花哨、颜色干净**

设计风格三大类：

| 类别                 | 权重 | 说明                       |
| -------------------- | ---- | -------------------------- |
| **Minimal 极简**     | 40%  | 黑白灰、点线面、留白       |
| **Digital 数字科技** | 40%  | 大数字、霓虹、网格、HUD    |
| **Geo 几何布局**     | 20%  | 方块、分割线、人工对齐美学 |

# 📌 **全年视觉体系（男性专属）**

### 🔲 色系

- 黑、白、灰
- 深蓝、冷青、冷绿
- 黑橙、黑黄、黑红（高对比）
- 金属银、金属灰、铝色质感

### 🔳 设计语言

- 大号数字（Massive Typography）
- 线条分区
- 网格、矩阵、点阵
- HUD 科技光
- 留白 + 单一主元素

### 🔧 男性用户最强购买点

1. **可读性强**
2. **看起来“专业 / 高级 / 冷静”**
3. **不花、不乱、不用图案**
4. **黑色背景最耐看、最耐久**
