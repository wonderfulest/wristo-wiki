   # 📌 **全年策略（3 大原则）**

   ### **1）保持月更 4 款（基础节奏）**

   全球风景赛道最吃“稳定输出”，全年保持：

   - **3 款氛围风景类**
   - **1 款主题增强版（更高级）**

   ### **2）所有节日用“自然氛围”替代，不做具体节日元素**

   例如：

   - 圣诞 → 冬季雪景
   - 情人节 → 粉色花海 / 日落
   - 万圣节 → 紫橙夜景

   避免宗教、政治，依然匹配用户季节情绪。

   ### **3）三个风格比重（全年保持黄金比例）**

   | 风格方向              | 比例 | 说明                   |
   | --------------------- | ---- | ---------------------- |
   | **自然风景**          | 50%  | 山、湖、森林、星空     |
   | **城市摄影**          | 30%  | 夜景、街道、建筑线条   |
   | **旅行 / 高审美构图** | 20%  | 海岸线、徒步路线、岛屿 |

   保证多样性 & 高审美。
