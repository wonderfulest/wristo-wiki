# 自定义颜色

支持自定义选择颜色，

支持的颜色选择项包括：

- 小时颜色
- 分钟颜色
- 冒号颜色 colon
- 日期颜色
- 图标颜色
- 数值颜色
- 背景颜色
- 强调颜色
- 

这些颜色来自 佳明的API ： https://developer.garmin.com/connect-iq/api-docs/Toybox/Graphics.html
