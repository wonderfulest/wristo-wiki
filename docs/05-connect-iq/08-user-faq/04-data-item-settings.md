# 指示器提示类



**用户FAQ**

## 蓝牙

表示蓝牙是否连接。显示规则，如果手表与手机连接，则亮色显示，否则，暗色显示。

保证连接状态需要满足下面两个条件：

1. 手机蓝牙打开，并配对成功
2. Connect APP 打开后，右上角的手表图标呈现绿色小点；并且 Connect APP 处于后台时，应该是运行状态，保证进程不被手机杀掉；

## 消息通知

手机上的消息通知数量。指示器类型显示规则：如果条数大于0，亮色显示，如果条数为0，则暗色显示。

## 闹钟数量

手表上设置的闹钟数量。指示器类型显示规则：如果条数大于0，亮色显示，如果条数为0，则暗色显示。

## 勿扰时间

表示当前是否处于 勿扰时间，为用户设置的夜间休息时间。

**开发者关注部分**

总共有 4种指示器类型

```
// indicator
:INDICATOR_TYPE_BLUETOOTH, // 21 蓝牙连接状态
:INDICATOR_TYPE_ALARM, // 22 闹钟 （indicator不支持数量，所以用indicator,否则用field）
:INDICATOR_TYPE_NOTIFICATIONS, // 23 通知 （indicator不支持数量，所以用indicator,否则用field）
:INDICATOR_TYPE_DISTURB, // 24 勿扰时间
```

当 闹钟 和 通知，支持条数显示的时候，使用 下面的 Symbol 进行配置。  

```
:FIELD_TYPE_NOTIFICATIONS, // 19 通知
:FIELD_TYPE_ALARMS, // 20 闹钟 
```







## SpO₂ 

血氧饱和度

**Pulse-Ox**（脉搏血氧，Pulse Oximetry 的缩写）是一种用于测量血氧饱和度（SpO₂）和脉搏率的技术。它通常通过光学传感器（光电血氧检测）来估算血液中的氧气含量，常见于智能手表、智能手环和医疗设备。

### **工作原理**

Pulse-Ox 传感器通常使用红光和红外光，通过皮肤照射到血管，并根据血液对不同波长光的吸收情况，计算血氧饱和度（SpO₂）。

### **在智能手表中的应用**

Garmin、Apple Watch、Fitbit 等智能手表品牌都配备了 Pulse-Ox 传感器，用于监测：

- **血氧饱和度（SpO₂）**：帮助评估血液携带氧气的能力。
- **睡眠监测**：追踪夜间血氧水平，识别睡眠呼吸暂停等问题。
- **高原适应性**：在登山或高海拔环境下，帮助评估身体对氧气稀薄环境的适应情况。

如果你是想在 Garmin 设备上使用或开发与 Pulse-Ox 相关的功能，可以查阅 Garmin 官方 Connect IQ SDK 文档，看看是否有相关的 API 支持。



## VO₂ Max

**VO₂ Max**（最大摄氧量，最大氧气摄取量）是指人体在剧烈运动时每分钟能够摄取和使用的最大氧气量，通常以 **ml/kg/min（毫升/千克/分钟）** 为单位。它是衡量**有氧耐力**（心肺耐力）和**心血管健康水平**的关键指标。

### **VO₂ Max 的意义**

- **衡量心肺耐力**：VO₂ Max 越高，代表心肺系统输送氧气到肌肉的能力越强，耐力更好。
- **运动表现预测**：耐力运动（如跑步、骑行）选手的 VO₂ Max 较高，通常表现更优。
- **健康评估**：较低的 VO₂ Max 可能与心血管疾病、代谢综合征等健康问题相关。

### **如何测量 VO₂ Max**

VO₂ Max 可以通过 **实验室测试** 或 **智能设备估算** 来测得：

1. **实验室测试**（最精准）
   - 需要佩戴面罩，在跑步机或功率自行车上进行递增负荷测试。
   - 通过测量吸入和呼出的氧气与二氧化碳含量，得出 VO₂ Max 值。
2. **智能设备估算**（近似值）
   - Garmin、Apple Watch、Polar 等智能手表利用**心率、步频、运动数据**，结合算法估算 VO₂ Max。
   - 主要基于跑步或骑行时的心率变化、速度、功率等数据。

### **VO₂ Max 标准值**

VO₂ Max 受 **年龄、性别、训练水平** 影响，不同人群的参考范围如下（单位：ml/kg/min）：

| 年龄  | 男性（优秀） | 女性（优秀） |
| ----- | ------------ | ------------ |
| 20-29 | 50+          | 45+          |
| 30-39 | 45+          | 40+          |
| 40-49 | 40+          | 35+          |
| 50+   | 35+          | 30+          |

> 业余跑者 VO₂ Max **40-50 ml/kg/min**，职业运动员可能高达 **70-90 ml/kg/min**（如顶级马拉松选手）。

### **如何提升 VO₂ Max**

1. **高强度间歇训练（HIIT）**：短时间高强度冲刺训练，提高心肺功能。
2. **有氧耐力训练**：如长距离跑步、骑行、游泳，增强心脏和肺活量。
3. **力量训练**：提升肌肉代谢效率，间接优化有氧耐力。
4. **高原训练**：在低氧环境下训练，可增强血液携氧能力（专业运动员常用）。
