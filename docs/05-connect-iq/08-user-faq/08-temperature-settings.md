# Temperature settings

## Celsius or Fahrenheit ?

You can select on your device or in the Connect app, the location is at: Settings -> Format -> Units -> Temperature.

## Connect IQ App Settings page

In the settings page, you can configure the following options:

### Temperature Unit

When the switch is turned off, no unit is displayed; when turned on, Celsius is represented by 'C,' and Fahrenheit is represented by 'F.'"

### Temperature type

Weather Forecast(outdoor): Currently, only Garmin's built-in weather API is supported. If displayed as '--,' it indicates that the current device is not supported.

Sensor temperature(indoor): Display the measured values of the Garmin device's temperature sensor. If displayed as '--,' it indicates that the current device is not supported.
