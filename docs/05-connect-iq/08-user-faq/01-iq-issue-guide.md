### FAQ: IQ 错误处理指南

#### Q1: 遇到 IQ 错误时该如何处理？

A1: 如果您的 Garmin 设备上出现 IQ 错误，请按照以下步骤操作：

1. 连接您的 Garmin 设备到电脑。
2. 进入设备存储目录 `/Garmin/Apps/Logs/`。
3. 找到日志文件 `CIQ_LOG.bak`。
4. 将该日志文件发送至邮箱 **wonder.gface@gmail.com**，并在邮件中简要描述您遇到的问题。

#### Q2: 发送日志文件后，我需要等待多久？

A2: 我们通常会在 **1-3 个工作日** 内查看您的日志文件，并提供解决方案或后续指导。

#### Q3: 如果找不到 `CIQ_LOG.bak` 文件怎么办？

A3: 请确认您的 Garmin 设备已连接至电脑，并确保该路径存在。如果仍然找不到，请尝试重新启动设备，并在运行应用后再次检查该目录。

#### Q4: 是否有其他方式获取日志文件？

A4: 部分设备可能存储日志文件的方式不同。如果 `CIQ_LOG.bak` 文件缺失，请检查 `/Garmin/Apps/Logs/` 目录下是否有其他 `.bak` 或 `.log` 结尾的文件，并一并发送给我们。

#### Q5: 发送日志时，需要提供哪些额外信息？

A5: 请在邮件中附上以下信息，以帮助我们更快地分析问题：

- Garmin 设备型号（例如 Fenix 7、Forerunner 955 等）。
- 安装的表盘或应用名称（如果适用）。
- 遇到 IQ 错误的具体情况（例如何时发生、是否有特定操作触发）。

如果您有任何其他问题，欢迎随时联系我们！

### FAQ: IQ Error Troubleshooting Guide

#### Q1: What should I do if I encounter an IQ error?

A1: If you experience an IQ error on your Garmin device, please follow these steps:

1. Connect your Garmin device to a computer.
2. Navigate to the directory `/Garmin/Apps/Logs/`.
3. Locate the log file `CIQ_LOG.bak`.
4. Send the log file to **wonder.gface@gmail.com**, along with a brief description of the issue.

#### Q2: How long will it take to get a response after sending the log file?

A2: We typically review log files and provide solutions or further guidance within **1-3 business days**.

#### Q3: What if I can’t find the `CIQ_LOG.bak` file?

A3: Please ensure your Garmin device is properly connected to your computer and that the directory exists. If the file is still missing, try restarting your device and check the directory again after running the app.

#### Q4: Are there alternative ways to retrieve the log file?

A4: Some devices may store log files differently. If `CIQ_LOG.bak` is missing, check the `/Garmin/Apps/Logs/` directory for other files with `.bak` or `.log` extensions and send them to us as well.

#### Q5: What additional information should I provide when sending the log file?

A5: To help us diagnose the issue faster, please include the following details in your email:

- Your Garmin device model (e.g., Fenix 7, Forerunner 955, etc.).
- The name of the watch face or app installed (if applicable).
- A brief description of the issue (e.g., when it occurred, any specific actions that triggered it).

If you have any other questions, feel free to contact us!
